# Security Policy

## Supported Versions

Currently, the latest version of DockTUI is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.4.x   | :white_check_mark: |
| < 1.4.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within DockTUI, please do not disclose it publicly. Instead, open a draft security advisory on GitHub or email the maintainer directly.

We will endeavor to respond to your report within 48 hours and work with you to release a patch as quickly as possible.
